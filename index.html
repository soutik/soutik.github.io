Projects

Twitter Sentiment Analysis with WordCloud

In this project I used TwitteR package inside Twitter to pull tweets containing a certain word/phrase and then used 'Sentiment' function inside R to analyse each tweet and attaching an emotion that the tweet showcases. We then create a wordcloud including the most frequent words found in those tweets.

This project can be used to find any word and any number of tweets (As permitted by the Twitter API by just changing 2 words in the code).

Find the code and output here : Twitter Sentiment Analysis
